# Redhope

**Live Server:** [https://testing-gold-seven.vercel.app](https://testing-gold-seven.vercel.app)

## About the Project
Redhope is a modern, full-stack web application built with Next.js and React to deliver a highly performant and seamless user experience. It features robust backend integration with MongoDB for data storage and Better Auth for secure user management. The platform also integrates Stripe for seamless payment processing, all wrapped in a beautiful, responsive user interface powered by HeroUI and Tailwind CSS.

## Features
- **Next.js Foundation:** Fast, server-side rendered, and highly optimized routing system.
- **Secure Authentication:** Comprehensive user management using Better Auth and MongoDB.
- **Payment Processing:** Integrated with Stripe for secure and reliable transactions.
- **Modern UI Design:** Beautifully styled components using Tailwind CSS v4 and HeroUI.
- **Rich Iconography:** Engaging visual elements provided by Lucide React and React Icons.
